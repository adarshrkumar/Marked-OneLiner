# Marked-OneLiner

A minimal Markdown-to-HTML converter in a single line of HTML.

## Usage

1. Open the page in a browser
2. Enter Markdown when prompted
3. View the rendered HTML in the page and copy from the prompt

## How It Works

Uses the [Marked](https://github.com/markedjs/marked) library to parse Markdown and convert it to HTML. The entire app is contained in one line of HTML.

## Tech Stack

- [Marked](https://github.com/markedjs/marked) via CDN
- Vanilla JavaScript
- Hosted via GitHub Pages
